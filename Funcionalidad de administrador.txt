-------------------------------------------------------------------------------------------------------------------
Funcionalidad de usuario: 
- Ver los datos de el personal del ateneo y filtrarlos mediante diferentes opciones

Funcionalidad de administrador: 

- Añadir nuevo empleado
- Eliminar un empleado
- Modificar un empleado


Documentación técnica: 

Página principal: index.php - Encontramos la tabla de personal y los filtros, depende de /includes/conexion.php

footer.html: Contiene el pie de pagina de la web
--------------------------------------------------------------------------------

Páginas de administrador: 
--------------------------------------------------------------------------------
/admin - Contiene diferentes archivos de la parte de administrador
- add_employee.php: Contiene funciones para añadir nuevos empleados

- edit_employee.php: Contiene funciones para editar los empleados existentes

- login.php: Contiene el formulario de inicio de sesion y la verificación de inicio de sesión, depende
             del archivo /includes/auth.php para funcionar

- logout.php: Contiene funciones para desloguear un usuario en la parte administrativa

indexadminpanel.php: Es la página principal con los paneles de administración de las tablas


/includes - 
- auth.php: Contiene las funciones de verificaciónde roles de usuario y sesion iniciada, depende de conexión.php
            para funcionar

- conexion.php: Contiene la conexión a la base de datos desplegada localmente


/directorio.sql
 Contiene un dump de la base de datos para desplegar de forma local
